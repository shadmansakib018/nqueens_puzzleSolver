N-Queens Puzzle

The eight queens puzzle is the problem of placing eight chess queens on an 8×8 chessboard so that no two queens threaten each other. Therefore a solution requires that no two queens share the same row, column, or diagonal. The eight queens puzzle is an example of the more complicated n queens problem of placing n non-attacking queens on an n×n chessboard. Solutions exist for all natural numbers n with the exception of n = 2 and n = 3.
